Framework

The Two Pillars of Evaluation

Every evaluation is built on two complementary components, each serving a distinct purpose.

1

Role-Based Skill Competency

70% of evaluation weight

Each position has a defined skill matrix, observable performance levels (1–5), minimum stability thresholds, and senior-track thresholds. Skills are tied directly to the responsibilities of the role.

  • Designers are evaluated on documentation and meeting discipline
  • CAs are evaluated on technical authority and code literacy
  • PMs are evaluated on ownership and coordination

Evaluation is role-specific — not generic.

2

Core Values Alignment

30% of evaluation weight

Core values reflect how work is executed. Values do not override skill gaps. Skills determine role readiness. Values determine trust and leadership suitability.

Both matter. They serve different purposes.

How Managers Should Use This Tool

✓ Managers Should

  • Conduct structured quarterly evaluations
  • Identify areas below stability threshold
  • Trigger Deficiency Correction Plans when required
  • Align authority with demonstrated capability
  • Guide development conversations

✗ Managers Should Avoid

  • Inflating scores to avoid difficult conversations
  • Using the form as a surprise mechanism
  • Completing reviews without prior discussion
  • Treating the form as a compliance exercise
The value is in the conversation and follow-through.
Correction

The Deficiency Correction Plan

When a skill falls below threshold, the system triggers a structured correction process. This is structured coaching — not punishment. The objective is measurable growth within 30–60 days.

1

Gap Identification

System automatically flags skills below stability threshold and identifies the specific gap.

2

Root Cause Classification

Manager determines whether the gap is experience-based, training-based, or a role-fit concern.

3

Defined Corrective Actions

Specific, actionable steps are documented with clear ownership and expectations.

4

Measurable Target

A specific skill level target is set with a 30-day checkpoint for progress review.

5

Timeline & Accountability

Follow-up date locked in. Authority temporarily aligned to current demonstrated capability.

6

Review & Reassess

If gap persists across multiple cycles, role fit must be evaluated. Not everyone fits every role long-term — that is normal in scaling organizations.

Advancement

Promotion Readiness

Promotions are unlocked — not negotiated.

Each role has defined skill thresholds, required stability levels, and senior-track criteria. This ensures fairness and protects team morale.

✓

Majority at Target Level

Most required skills must be at the defined target level for the next role.

⊘

No Critical Gaps

No skill can be below the minimum threshold for the target position.

↻

Consistent Performance

Sustained performance across multiple evaluation cycles, not a single snapshot.

⚑

No Disqualifiers

Universal disqualifiers (integrity, repeated deficiencies) automatically block promotion.
